Array#bsearch

  • - returns an Enumerator when not passed a block
  • - raises a TypeError if the block returns an Object (FAILED - 1)
  • - raises a TypeError if the block returns a String (FAILED - 2)

Array#bsearch when no block is given returned Enumerator

  • - size returns nil (ERROR - 3)

Array#bsearch with a block returning true or false

  • - returns nil if the block returns false for every element
  • - returns nil if the block returns nil for every element
  • - returns element at zero if the block returns true for every element
  • - returns the element at the smallest index for which block returns true

Array#bsearch with a block returning negative, zero, positive numbers

  • - returns nil if the block returns less than zero for every element
  • - returns nil if the block returns greater than zero for every element
  • - returns nil if the block never returns zero
  • - accepts (+/-)Float::INFINITY from the block
  • - returns an element at an index for which block returns 0.0 (FAILED - 4)
  • - returns an element at an index for which block returns 0

Array#bsearch with a block that calls break

  • - returns nil if break is called without a value
  • - returns nil if break is called with a nil value
  • - returns object if break is called with an object

  1. Array#bsearch raises a TypeError if the block returns an Object FAILED

    Expected TypeError but no exception was raised (#@result was returned)
