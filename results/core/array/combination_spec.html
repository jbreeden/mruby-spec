Array#combination

  • - returns an enumerator when no block is provided (ERROR - 1)
  • - returns self when a block is given (ERROR - 2)
  • - yields nothing for out of bounds length and return self (ERROR - 3)
  • - yields the expected combinations (ERROR - 4)
  • - yields nothing if the argument is out of bounds (ERROR - 5)
  • - yields a copy of self if the argument is the size of the receiver (ERROR - 6)
  • - yields [] when length is 0 (ERROR - 7)
  • - yields a partition consisting of only singletons (ERROR - 8)
  • - generates from a defensive copy, ignoring mutations (ERROR - 9)
