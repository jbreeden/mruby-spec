Array#join

  • - returns a string formed by concatenating each element.to_str separated by separator
  • - uses the same separator with nested arrays
  • - returns an empty string if the Array is empty
  • - returns a US-ASCII string for an empty Array (ERROR - 1)
  • - returns a string formed by concatenating each String element separated by $, (FAILED - 2)
  • - attempts coercion via #to_str first
  • - attempts coercion via #to_ary second
  • - attempts coercion via #to_s third
  • - raises a NoMethodError if an element does not respond to #to_str, #to_ary, or #to_s (ERROR - 3)
  • - raises an ArgumentError when the Array is recursive
  • - taints the result if the Array is tainted and non-empty (ERROR - 4)
  • - does not taint the result if the Array is tainted but empty (ERROR - 5)
  • - taints the result if the result of coercing an element is tainted (ERROR - 6)
  • - taints the result if the result of coercing an element is tainted (ERROR - 7)
  • - untrusts the result if the Array is untrusted and non-empty (ERROR - 8)
  • - does not untrust the result if the Array is untrusted but empty (ERROR - 9)
  • - untrusts the result if the result of coercing an element is untrusted (ERROR - 10)
  • - untrusts the result if the result of coercing an element is untrusted (ERROR - 11)
  • - uses the first encoding when other strings are compatible (ERROR - 12)
  • - uses the widest common encoding when other strings are incompatible (ERROR - 13)
  • - fails for arrays with incompatibly-encoded strings (ERROR - 14)
  • - does not separate elements when the passed separator is nil
  • - calls #to_str to convert the separator to a String
  • - does not call #to_str on the separator if the array is empty (ERROR - 15)
  • - does not call #to_str on the separator if the array is empty (ERROR - 16)
  • - raises a TypeError if the separator cannot be coerced to a String by calling #to_str
  • - raises a TypeError if passed false as the separator

Array#join with a tainted separator

  • - does not taint the result if the array is empty (ERROR - 17)
  • - does not taint the result if the array has only one element (ERROR - 18)
  • - taints the result if the array has two or more elements (ERROR - 19)

Array#join with an untrusted separator

  • - does not untrust the result if the array is empty (ERROR - 20)
  • - does not untrust the result if the array has only one element (ERROR - 21)
  • - untrusts the result if the array has two or more elements (ERROR - 22)

Array#join with $,

  • - separates elements with default separator when the passed separator is nil (FAILED - 23)
