Array#keep_if

  • - returns the same array if no changes were made
  • - deletes elements for which the block returns a false value
  • - returns an enumerator if no block is given

Array#keep_if when no block is given returned Enumerator

  • - size returns the enumerable size (ERROR - 1)

Array#keep_if on frozen objects

  • - returns an Enumerator if no block is given (ERROR - 2)

Array#keep_if on frozen objects with truthy block

  • - keeps elements after any exception (ERROR - 3)
  • - raises a RuntimeError (ERROR - 4)

Array#keep_if on frozen objects with falsy block

  • - keeps elements after any exception (ERROR - 5)
  • - raises a RuntimeError (ERROR - 6)
