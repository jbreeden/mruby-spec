Array#map

  • - returns a copy of array with each element replaced by the value returned by block
  • - does not return subclass instance
  • - does not change self
  • - returns the evaluated value of block if it broke in the block
  • - returns an Enumerator when no block given
  • - does not copy tainted status (ERROR - 1)
  • - does not copy untrusted status (ERROR - 2)

Array#map when no block is given returned Enumerator

  • - size returns the enumerable size (ERROR - 3)

Array#map!

  • - replaces each element with the value returned by block
  • - returns self
  • - returns the evaluated value of block but its contents is partially modified, if it broke in the block
  • - returns an Enumerator when no block given, and the enumerator can modify the original array
  • - keeps tainted status (ERROR - 4)
  • - keeps untrusted status (ERROR - 5)

Array#map! when frozen

  • - raises a RuntimeError (ERROR - 6)
  • - raises a RuntimeError when empty (ERROR - 7)
  • - raises a RuntimeError when calling #each on the returned Enumerator (ERROR - 8)
  • - raises a RuntimeError when calling #each on the returned Enumerator when empty (ERROR - 9)

Array#map! when no block is given returned Enumerator

  • - size returns the enumerable size (ERROR - 10)
