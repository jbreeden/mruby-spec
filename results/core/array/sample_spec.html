Array#sample

  • - returns nil for an empty Array
  • - returns a single value when not passed a count
  • - returns an empty Array when passed zero
  • - returns an Array of elements when passed a count
  • - returns elements from the Array
  • - returns at most the number of elements in the Array
  • - does not return the same value if the Array has unique values
  • - may return the same value if the array is not unique
  • - calls #to_int to convert the count when passed an Object
  • - raises ArgumentError when passed a negative count
  • - does not return subclass instances with Array subclass
