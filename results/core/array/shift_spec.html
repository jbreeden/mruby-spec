Array#shift

  • - removes and returns the first element
  • - returns nil when the array is empty
  • - properly handles recursive arrays
  • - raises a RuntimeError on a frozen array (ERROR - 1)
  • - raises a RuntimeError on an empty frozen array (ERROR - 2)

Array#shift passed a number n as an argument

  • - removes and returns an array with the first n element of the array (FAILED - 3)
  • - does not corrupt the array when shift without arguments is followed by shift with an argument (FAILED - 4)
  • - returns a new empty array if there are no more elements (FAILED - 5)
  • - returns whole elements if n exceeds size of the array (FAILED - 6)
  • - does not return self even when it returns whole elements
  • - raises an ArgumentError if n is negative (FAILED - 7)
  • - tries to convert n to an Integer using #to_int (FAILED - 8)
  • - raises a TypeError when the passed n can be coerced to Integer (FAILED - 9)
  • - raises an ArgumentError if more arguments are passed (FAILED - 10)
  • - does not return subclass instances with Array subclass (FAILED - 11)
  • - returns an untainted array even if the array is tainted (ERROR - 12)
  • - keeps taint status (ERROR - 13)
