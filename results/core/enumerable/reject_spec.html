Enumerable#reject

  • - returns an array of the elements for which block is false
  • - returns an Enumerator if called without a block (ERROR - 1)
  • - gathers whole arrays as elements when each yields multiple (FAILED - 2)

Enumerable#reject Enumerable with size when no block is given returned Enumerator

  • - size returns the enumerable size (ERROR - 3)

  • - size returns nil (ERROR - 4)
