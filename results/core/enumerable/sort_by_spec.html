Enumerable#sort_by

  • - returns an array of elements ordered by the result of block
  • - sorts the object by the given attribute
  • - returns an Enumerator when a block is not supplied
  • - gathers whole arrays as elements when each yields multiple

Enumerable#sort_by Enumerable with size when no block is given returned Enumerator

  • - size returns the enumerable size (ERROR - 1)

Enumerable#sort_by Enumerable with no size when no block is given returned Enumerator

  • - size returns nil (ERROR - 2)
