Float#%

  • - returns self modulo other
  • - returns self when modulus is +Infinity
  • - returns -Infinity when modulus is -Infinity
  • - returns NaN when called on NaN or Infinities
  • - returns NaN when modulus is NaN
  • - returns -0.0 when called on -0.0 with a non zero modulus
  • - raises a ZeroDivisionError if other is zero (ERROR - 1)

Float#modulo

  • - returns self modulo other (ERROR - 2)
  • - returns self when modulus is +Infinity (ERROR - 3)
  • - returns -Infinity when modulus is -Infinity (ERROR - 4)
  • - returns NaN when called on NaN or Infinities (ERROR - 5)
  • - returns NaN when modulus is NaN (ERROR - 6)
  • - returns -0.0 when called on -0.0 with a non zero modulus (ERROR - 7)
  • - raises a ZeroDivisionError if other is zero (ERROR - 8)
