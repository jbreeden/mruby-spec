ObjectSpace.each_object

  • - calls the block once for each living, non-immediate object in the Ruby process (ERROR - 1)
  • - calls the block once for each class, module in the Ruby process (ERROR - 2)
  • - returns an enumerator if not given a block (ERROR - 3)
  • - finds an object stored in a global variable (ERROR - 4)
  • - finds an object stored in a top-level constant (ERROR - 5)
  • - finds an object stored in a second-level constant (ERROR - 6)
  • - finds an object stored in a local variable (ERROR - 7)
  • - finds an object stored in a local variable captured in a block explicitly (ERROR - 8)
  • - finds an object stored in a local variable captured in a block implicitly (ERROR - 9)
  • - finds an object stored in a local variable captured in by a method defined with a block (ERROR - 10)
  • - finds an object stored in a local variable captured in a Proc#binding (ERROR - 11)
  • - finds an object stored in a local variable captured in a Kernel#binding (ERROR - 12)
  • - finds an object stored in a local variable set in a binding manually (ERROR - 13)
  • - finds an object stored in an array (ERROR - 14)
  • - finds an object stored in a hash key (ERROR - 15)
  • - finds an object stored in a hash value (ERROR - 16)
  • - finds an object stored in an instance variable (ERROR - 17)
  • - doesn't find an object stored in a WeakRef that should have been cleared (ERROR - 18)
  • - finds an object stored in a thread local (ERROR - 19)
  • - finds an object stored in a fiber local (ERROR - 20)
  • - finds an object captured in an at_exit handler (ERROR - 21)
  • - finds an object captured in finalizer (ERROR - 22)
  • - walks a class and its normal descendants when passed the class's singleton class (ERROR - 23)
