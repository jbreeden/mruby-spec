String#<<

  • - concatenates the given argument to self and returns self
  • - converts the given argument to a String using to_str
  • - raises a TypeError if the given argument can't be converted to a String
  • - raises a RuntimeError when self is frozen
  • - returns a String when given a subclass instance
  • - returns an instance of same class when called on a subclass
  • - taints self if other is tainted (ERROR - 1)
  • - untrusts self if other is untrusted (ERROR - 2)

String#<< with Integer

  • - concatencates the argument interpreted as a codepoint (ERROR - 3)
  • - returns a ASCII-8BIT string if self is US-ASCII and the argument is between 128-255 (inclusive) (ERROR - 4)
  • - raises RangeError if the argument is an invalid codepoint for self's encoding (ERROR - 5)
  • - raises RangeError if the argument is negative (ERROR - 6)
  • - doesn't call to_int on its argument
  • - raises a RuntimeError when self is frozen (ERROR - 7)

String#<< when self is in an ASCII-incompatible encoding incompatible with the argument's encoding

  • - uses self's encoding if both are empty (ERROR - 8)
  • - uses self's encoding if the argument is empty (ERROR - 9)
  • - uses the argument's encoding if self is empty (ERROR - 10)
  • - raises Encoding::CompatibilityError if neither are empty (ERROR - 11)

String#<< when the argument is in an ASCII-incompatible encoding incompatible with self's encoding

  • - uses self's encoding if both are empty (ERROR - 12)
  • - uses self's encoding if the argument is empty (ERROR - 13)
  • - uses the argument's encoding if self is empty (ERROR - 14)
  • - raises Encoding::CompatibilityError if neither are empty (ERROR - 15)

String#<< when self and the argument are in different ASCII-compatible encodings

  • - uses self's encoding if both are ASCII-only (ERROR - 16)
  • - uses self's encoding if the argument is ASCII-only (ERROR - 17)
  • - uses the argument's encoding if self is ASCII-only (ERROR - 18)
  • - raises Encoding::CompatibilityError if neither are ASCII-only (ERROR - 19)

String#<< when self is ASCII-8BIT and argument is US-ASCII

  • - uses ASCII-8BIT encoding (ERROR - 20)
