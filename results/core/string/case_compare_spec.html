String#===

  • - returns true if self <=> string returns 0
  • - returns false if self <=> string does not return 0
  • - ignores encoding difference of compatible string (ERROR - 1)
  • - considers encoding difference of incompatible string (ERROR - 2)
  • - considers encoding compatibility (ERROR - 3)
  • - ignores subclass differences
  • - returns false if obj does not respond to to_str
  • - returns obj == self if obj responds to to_str (FAILED - 4)
  • - returns obj == self if obj responds to to_str (ERROR - 5)
  • - is not fooled by NUL characters
