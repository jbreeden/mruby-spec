String#initialize with no arguments

String#initialize with an argument

  • - returns self
  • - replaces the content of self with other
  • - taints self if other is tainted (ERROR - 2)
  • - does not untaint self if other is untainted (ERROR - 3)
  • - untrusts self if other is untrusted (ERROR - 4)
  • - does not trust self if other is trusted (ERROR - 5)
  • - replaces the encoding of self with that of other (ERROR - 6)
  • - carries over the encoding invalidity (ERROR - 7)
  • - tries to convert other to string using to_str
  • - raises a TypeError if other can't be converted to string
  • - raises a RuntimeError on a frozen instance that is modified
  • - raises a RuntimeError on a frozen instance when self-replacing
