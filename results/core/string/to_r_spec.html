String#to_r

  • - returns a Rational object (ERROR - 1)
  • - returns (0/1) for the empty String (ERROR - 2)
  • - returns (n/1) for a String starting with a decimal _n_ (ERROR - 3)
  • - ignores trailing characters (ERROR - 4)
  • - ignores leading spaces (ERROR - 5)
  • - does not ignore arbitrary, non-numeric leading characters (ERROR - 6)
  • - treats leading hypens as minus signs (ERROR - 7)
  • - does not treat a leading period without a numeric prefix as a decimal point (ERROR - 8)
  • - understands decimal points (ERROR - 9)
  • - ignores underscores between numbers (ERROR - 10)
  • - understands a forward slash as separating the numerator from the denominator (ERROR - 11)
  • - returns (0/1) for Strings it can't parse (ERROR - 12)
