The module keyword

  • - creates a new module without semicolon (ERROR - 1)
  • - creates a new module with a non-qualified constant name (ERROR - 2)
  • - creates a new module with a qualified constant name
  • - creates a new module with a variable qualified constant name
  • - reopens an existing module (ERROR - 3)
  • - reopens a module included in Object (ERROR - 4)
  • - raises a TypeError if the constant is a Class
  • - raises a TypeError if the constant is a String
  • - raises a TypeError if the constant is a Fixnum
  • - raises a TypeError if the constant is nil
  • - raises a TypeError if the constant is true
  • - raises a TypeError if the constant is false

Assigning an anonymous module to a constant

  • - sets the name of the module (ERROR - 5)
  • - does not set the name of a module scoped by an anonymous module (ERROR - 6)
  • - sets the name of contained modules when assigning a toplevel anonymous module (ERROR - 7)
