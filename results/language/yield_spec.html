  • - raises a LocalJumpError (ERROR - 3)

  • - passes an empty Array when the argument is an empty Array
  • - passes nil as a value
  • - passes a single value
  • - passes a single, multi-value Array

The yield call taking a single argument yielding to a lambda

  • - passes an empty Array when the argument is an empty Array
  • - passes nil as a value
  • - passes a single value
  • - passes a single, multi-value Array

The yield call taking multiple arguments

  • - raises a LocalJumpError when the method is not passed a block (ERROR - 4)
  • - passes the arguments to the block

The yield call taking a single splatted argument

  • - raises a LocalJumpError when the method is not passed a block (ERROR - 5)

  • - passes the value when the argument is an Array containing a single value

  • - passes the element of a single element Array
  • - passes no values when give nil as an argument

  • - does not pass an argument value if the splatted argument is nil
